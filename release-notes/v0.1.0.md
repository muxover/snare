# Snare v0.1.0 — Initial release

**Release date:** 2026-02-28

### Install

```bash
go install github.com/muxover/snare@v0.1.0
```

---

## Summary

First release of **Snare**: an HTTP/HTTPS proxy CLI that intercepts, captures, and replays traffic. Single binary, supports HTTP/1.1 and HTTP/2, optional MITM for HTTPS, persistent capture storage, and export to HAR/JSON.

---

## What's included

### Core

- **HTTP/HTTPS proxy** — CONNECT support and HTTPS MITM via dynamic per-host certificates.
- **Protocols** — HTTP/1.1 and HTTP/2.
- **Capture** — Automatic capture of every request/response: method, URL, headers, body, status, duration. Persisted as JSON; export to HAR or JSON; replay and save commands.
- **CA** — Generate and install; bind to all interfaces (`--bind 0.0.0.0`); graceful shutdown on SIGINT/SIGTERM.

### CLI & config

- **Paths** — Default store `~/.snare/captures`, CA `~/.snare`; env: `SNARE_STORE`, `SNARE_CA`.
- **Serve** — `serve --verbose` / `-v` for debug logging; port validation (1–65535); clear error handling and exit codes.
- **Version** — `snare --version` reports 0.1.0.
