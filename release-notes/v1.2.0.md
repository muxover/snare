# Snare v1.2.0 — Better observability

**Release date:** 2026-05-02

### Install

```bash
go install github.com/muxover/snare@v1.2.0
```

## Summary

Adds live watching, richer listing (latency, time range, body search), capture comparison, and full-store scans when filters need every capture on disk.

## What's included

- `snare watch` — poll the store and print new captures (same columns as `list`; Ctrl+C to stop)
- `snare list` — duration column; `--since` / `--until` (RFC3339 or `2006-01-02`); `--body` to match request or response body
- `snare diff` — compare two captures by ID or prefix (request, response, headers, bodies)
- `AllFromDisk` on the capture store for unbounded list filters
