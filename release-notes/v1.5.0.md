# Snare v1.5.0 — Interception

**Release date:** 2026-05-02

### Install

```bash
go install github.com/muxover/snare@v1.5.0
```

## Summary

Adds active request interception. Run `serve --intercept` to pause matching requests mid-flight; then forward, edit, or drop them from a second terminal using the `snare intercept` commands.

## What's included

- `snare serve --intercept <pattern>` — intercept requests whose URL matches (use `*` for all traffic)
- `snare serve --intercept-timeout` — auto-drop after this duration if no decision is made (default 5m)
- `snare intercept list` — see all requests currently held by the proxy
- `snare intercept forward [id]` — release a held request to the origin unchanged
- `snare intercept edit [id]` — open in `$EDITOR`, modify method/headers/body, then forward
- `snare intercept drop [id]` — discard the request; client receives 502
- Works across plain HTTP, MITM HTTP/1.1, and MITM HTTP/2
- Pending requests stored as JSON in `~/.snare/intercept/` (overridable via `SNARE_INTERCEPT`)
