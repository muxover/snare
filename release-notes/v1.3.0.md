# Snare v1.3.0 — Proxy control

**Release date:** 2026-05-02

### Install

```bash
go install github.com/muxover/snare@v1.3.0
```

## Summary

Adds upstream proxy chaining, outbound rewrite/mutation rules on `serve`, URL-match batch replay, and HAR import. This release also updates key networking/compression dependencies.

## What's included

- `snare serve --upstream-proxy` to route outbound traffic through another proxy
- `snare serve --rewrite-host from=to` (repeatable) for outbound host rewrite rules
- `snare serve --add-header` and `--remove-header` (repeatable) for outbound header mutation
- `snare replay --match <substring>` to replay all matching captures
- `snare import [file.har]` to import HAR files into the capture store
- Dependency updates: `golang.org/x/net v0.53.0`, `github.com/andybalholm/brotli v1.2.1`
