# Snare v1.7.0 — WebSocket capture

**Release date:** 2026-05-13

### Install

```bash
go install github.com/muxover/snare@v1.7.0
```

## Summary

Snare can record WebSocket traffic end-to-end wherever the proxy terminates or originates the upgrade: MITM over HTTP/1.1 (`101`), MITM over HTTP/2 extended CONNECT (RFC 8441, `200` on the stream), and cleartext `http://` / `https://` proxy requests (`serveHTTP`) using dial + hijack so the origin socket stays open for the relay.

## What is included

- **MITM HTTP/1.1:** same as before — full `101` handshake in the capture, then `websocket.frames` when the session ends.
- **MITM HTTP/2:** detects `CONNECT` with `:protocol` websocket; responds with `200` on the HTTP/2 stream; reads RFC6455 frames from the client stream (unmasked on the wire per RFC 8441); opens an HTTP/1.1 WebSocket to the origin and relays in both directions with frame logging.
- **Cleartext proxy:** for WebSocket upgrades on absolute-form `http://` or `https://` requests, Snare dials the origin, completes the handshake, hijacks the client connection, writes `101`, and runs the same frame relay and capture path.
- `snare show` and HAR `_webSocketMessages` unchanged in shape from the first v1.7.0 drop.
