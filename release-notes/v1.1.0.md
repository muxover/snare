# Snare v1.1.0 — First release

**Release date:** 2026-03-11

### Install

```bash
go install github.com/muxover/snare@v1.1.0
```

## Summary

First release of Snare: an HTTP/HTTPS proxy CLI that intercepts, captures, and replays traffic. Set `HTTP_PROXY`/`HTTPS_PROXY` to the proxy and use the CLI to list, show, replay, save, and export captures (JSON or HAR).

## What's included

- `snare serve` — run proxy (HTTP + HTTPS MITM, HTTP/1.1 and HTTP/2)
- `snare list` — list captures with optional filters (method, status, url, host)
- `snare show` / `snare replay` — inspect and re-send requests
- `snare save` / `snare export` — save or export to JSON/HAR
- `snare clear` / `snare delete` — clear or delete captures
- `snare ca generate` / `snare ca install` — CA for HTTPS MITM
- Body decompression (gzip, deflate, brotli) for readable captures
- Config via `SNARE_STORE`, `SNARE_CA` and serve flags
